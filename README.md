HIV-1 genetic tropism genetic prediction for CRF01AE is lacking. While some tools are available, currently, PhenoSeq (Cashin K, et al., (2013) Retrovirology 10:24; https://tools.burnet.edu.au/phenoseq/) is the best tool. 

We set out to use machine-learning prediciton models on previously phenotypically determined V3 loop data (LANL) using recent analyses on determine R5 vs X4 tropism in CRF01AE. We explored this in Capoferri and Boltz et al. (manuscript in preparation) as part of a larger study. 

The set of scripts and data are meant to be an initial road map to building machine-learning models for other subtypes and to execute for any CRF01AE data that you may have.
